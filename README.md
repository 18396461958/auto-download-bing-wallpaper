# Bing Wallpaper


每日通过Github Actions自动爬取Bing壁纸

picurl.py会在当前文件夹创建一个DownloadedWallpapers文件夹,
并将下载的壁纸保存到该文件夹中。

Clone本仓库到本地后双击运行picurl.py即可爬取壁纸
